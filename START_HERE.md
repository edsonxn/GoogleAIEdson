## ⚡ Quick Start - Lector de Comentarios YouTube Live

### 🎯 En 3 pasos:

#### 1️⃣ Instalar pytchat
```powershell
pip install pytchat
```

#### 2️⃣ Iniciar servidor
```powershell
npm start
```

#### 3️⃣ Usar la función
- Abre: http://localhost:3000
- Haz clic en **"Comentarios del Live"** (barra lateral)
- Ingresa URL o ID de YouTube Live
- Haz clic en **"Iniciar Lectura"**
- 🎉 ¡Listo! Los comentarios aparecerán en tiempo real
